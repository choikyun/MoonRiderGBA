# MOON DEFENDER
GBAのゲームです。  
エミュレータか実機で遊べます。  

このソースは最新のDivkitProでビルドできます。

## 修正
2017-02-28　実機でも見やすいようにキャラクタを大きくしました。  

## 遊び方
よくあるミサイルコマンドです。

上空から降り注ぐミサイルを撃破してください。  
AMB（Anti-Missile-Battery）、もしくは都市がすべて破壊されるとゲームオーバーです。  
ゲームに終わりはないのでひたすらスコアを稼いでください。  

ミサイルを撃破した高度×残り都市数がスコアになります。（上空のほうがスコアは高いです）  
なお、都市を破壊された後の更地を爆撃されるとスコアが減算されます。  

### トロフィー
プレイ中、ある条件をクリアするとトロフィーがもらえます。  
トロフィーは6個あります。いくつ獲得したかはタイトル画面で確認できます。  

### 操作

**タイトル画面**

* **SELECT**　モード選択
* **START**　ゲーム開始
* **R+B**　ハイスコアクリア

**ゲーム画面**

* **A**　ミサイル発射
* **UP DOWN LEFT RIGHT**　照準移動
* **SELECT**　ポーズ

### キャラクタ

**アイテム**

* **補給**　残り弾数を回復
* **反応弾**　誘爆して複数のミサイルを撃破。ただしスコアにならない

**敵**

* **通常のミサイル**　都市を狙ってくる。レベルが上がるごとに速くなる。たまに弾頭が分裂。
* **UFO**　AMB目掛けて高速弾を撃ってくる。
